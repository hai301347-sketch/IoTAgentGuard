# IoT智能监控系统 - 接口文档

基础路径: `http://localhost:8080/api`
认证方式: 请求头 `Authorization: Bearer <token>`（登录/注册接口除外）

## 一、认证接口 `/api/auth`

### POST /api/auth/login
- 说明: 用户登录
- 请求体: `{ "username": "admin", "password": "admin123" }`
- 响应: `{ "code": 200, "data": { "token": "xxx", "username": "admin", "realName": "管理员", "role": "ADMIN" } }`

### POST /api/auth/register
- 说明: 用户注册
- 请求体: `{ "username": "user1", "password": "123456", "realName": "张三" }`
- 响应: `{ "code": 200, "message": "success" }`

## 二、设备接口 `/api/device`

### POST /api/device/add
- 说明: 新增设备
- 请求体: `{ "deviceName": "传感器A", "deviceCode": "D004", "deviceType": "传感器", "location": "101室", "description": "描述" }`

### PUT /api/device/update/{id}
- 说明: 更新设备信息

### DELETE /api/device/delete/{id}
- 说明: 删除设备

### GET /api/device/{id}
- 说明: 获取设备详情

### GET /api/device/list?page=1&size=10&keyword=xxx
- 说明: 分页查询设备
- 响应: `{ "code": 200, "data": { "list": [...], "total": 3, "page": 1, "size": 10 } }`

## 三、数据接口 `/api/data`

### POST /api/data/report
- 说明: 上报传感数据
- 请求体: `{ "deviceCode": "D001", "temperature": 25.5, "humidity": 60, "light": 500, "smoke": 10 }`

### GET /api/data/list?page=1&size=10&keyword=xxx
- 说明: 分页查询传感数据

### GET /api/data/device/{deviceId}?startTime=xxx&endTime=xxx
- 说明: 按设备查询历史数据

### GET /api/data/latest/{deviceId}
- 说明: 获取设备最新传感数据

## 四、告警接口 `/api/alert`

### GET /api/alert/list?page=1&size=10&keyword=xxx
- 说明: 分页查询告警记录

### PUT /api/alert/handle/{id}
- 说明: 处理告警（将status置为1）

## 五、仪表盘接口 `/api/dashboard`

### GET /api/dashboard/stats
- 说明: 获取仪表盘统计数据
- 响应: `{ "code": 200, "data": { "deviceTotal": 3, "onlineDeviceCount": 3, "todayDataCount": 100, "alertCount": 2, "temperatureAvg": 25.5, "humidityAvg": 60.0, "recentAlerts": [...], "hourlyData": [...] } }`

## 六、Agent接口 `/api/agent`

### POST /api/agent/simulator/start
- 说明: 启动数据模拟器

### POST /api/agent/simulator/stop
- 说明: 停止数据模拟器

### POST /api/agent/simulator/interval
- 说明: 设置采集频率
- 请求体: `{ "interval": 5 }`

### POST /api/agent/query
- 说明: 自然语言指令查询
- 请求体: `{ "command": "设置采集频率为5秒" }`
- 响应: `{ "code": 200, "data": "采集频率已设置为5秒" }`

### POST /api/agent/inspect
- 说明: 执行系统自动巡检
- 响应: 巡检报告文本

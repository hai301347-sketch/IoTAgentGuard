# IoT智能监控系统 - 实训项目开发报告

## 一、项目背景

本项目为物联网工程专业综合实训项目，旨在通过"AI Agent驱动型物联网设备智能监控管理系统"的完整开发，掌握前后端分离架构、数据库设计、物联网数据处理等核心技术。

## 二、项目目标

1. 掌握 Vue3 + Vite 前端项目开发流程
2. 掌握 Spring Boot 后端项目搭建与 RESTful API 设计
3. 理解 MySQL 数据库设计与 MyBatis 数据持久化
4. 实践 AI Agent 在物联网监控场景中的应用
5. 完成前后端联调与部署

## 三、技术选型

| 层次 | 技术 | 版本 | 选型理由 |
|------|------|------|----------|
| 前端框架 | Vue3 | 3.4 | Composition API，响应式编程 |
| 构建工具 | Vite | 5.0 | 极速热更新，开发体验好 |
| UI组件库 | Element Plus | 2.4 | 丰富的组件，适配后台管理 |
| 图表库 | ECharts | 5.4 | 数据可视化 |
| 状态管理 | Pinia | 2.1 | Vue3官方推荐 |
| HTTP客户端 | Axios | 1.6 | 拦截器，请求封装 |
| 后端框架 | Spring Boot | 2.7 | 快速开发，生态成熟 |
| ORM框架 | MyBatis | 2.3 | SQL灵活可控 |
| 数据库 | MySQL | 8.0 | 稳定可靠的关系型数据库 |
| 认证方案 | JWT | 0.9 | 无状态认证 |

## 四、系统功能

### 4.1 用户管理
- 用户登录（JWT Token认证）
- 用户注册
- 权限拦截

### 4.2 设备管理
- 设备增删改查
- 设备在线状态监控
- 分页搜索

### 4.3 数据监控
- 实时传感数据展示（温度、湿度、光照、烟雾）
- 历史数据查询与分页
- 数据图表可视化

### 4.4 告警管理
- 自动异常检测与告警
- 告警级别分类（CRITICAL/HIGH/MEDIUM）
- 告警处理状态跟踪

### 4.5 AI Agent功能
- 数据模拟器：定时生成模拟传感数据
- 告警检测器：自动检测异常数据
- 自然语言指令解析：通过文字控制采集
- 自动巡检：系统健康检查

## 五、开发过程总结

### 5.1 架构设计
采用经典三层架构（Controller-Service-Mapper），前后端完全分离，通过RESTful API通信。数据库设计遵循范式原则，4张核心表覆盖用户、设备、数据、告警四大业务实体。

### 5.2 核心难点
1. **JWT认证机制**：实现无状态的用户认证，拦截器统一处理
2. **MyBatis动态SQL**：XML中使用条件查询和分页
3. **AI Agent调度**：ScheduledExecutorService实现定时数据采集
4. **异常数据检测**：多维度阈值判断，分级告警
5. **前端实时刷新**：定时器驱动数据自动更新

### 5.3 创新点
1. 内置AI Agent模块，实现自动化数据采集与异常检测
2. 自然语言指令解析，提升系统交互体验
3. 自动巡检功能，主动发现问题并给出修复建议

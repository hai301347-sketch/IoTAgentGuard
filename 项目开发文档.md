# IoT智能监控系统 - 项目开发文档

## 一、项目概述

**项目名称：** IoT Agent驱动型物联网设备智能监控管理系统
**项目定位：** 面向校园实训与企业实习落地的物联网工程实战项目
**技术栈：** Vue3 + Vite（前端）| Spring Boot 2.7（后端）| MySQL 8.0（数据库）

## 二、系统架构

```
┌─────────────────────────────────────────────────────┐
│                   前端 Vue3 + Vite                    │
│  登录 │ 仪表盘 │ 设备管理 │ 实时监控 │ 历史数据 │ 告警 │
├─────────────────────────────────────────────────────┤
│                 HTTP API (RESTful)                    │
├─────────────────────────────────────────────────────┤
│              后端 Spring Boot 2.7                     │
│  Controller → Service → Mapper → MyBatis → MySQL    │
│  + JWT认证 + 全局异常处理 + 统一返回格式              │
│  + AI Agent模块（数据模拟/告警检测/NLP解析/自动巡检）  │
├─────────────────────────────────────────────────────┤
│                    MySQL 8.0                         │
│  sys_user │ iot_device │ env_data │ alert_record     │
└─────────────────────────────────────────────────────┘
```

## 四、数据库设计

### 4.1 用户表 (sys_user)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK | 主键 |
| username | VARCHAR(50) | 用户名（唯一） |
| password | VARCHAR(200) | 密码（BCrypt加密） |
| real_name | VARCHAR(50) | 真实姓名 |
| role | VARCHAR(20) | 角色（ADMIN/USER） |
| status | TINYINT | 状态（1启用/0禁用） |
| create_time | DATETIME | 创建时间 |
| update_time | DATETIME | 更新时间 |

### 4.2 设备信息表 (iot_device)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK | 主键 |
| device_name | VARCHAR(100) | 设备名称 |
| device_code | VARCHAR(50) | 设备编码（唯一） |
| device_type | VARCHAR(50) | 设备类型 |
| location | VARCHAR(200) | 安装位置 |
| status | VARCHAR(20) | 在线状态（ONLINE/OFFLINE） |
| description | VARCHAR(500) | 设备描述 |

### 4.3 环境传感数据表 (env_data)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK | 主键 |
| device_id | BIGINT | 关联设备ID |
| device_code | VARCHAR(50) | 设备编码 |
| temperature | DECIMAL(5,2) | 温度（°C） |
| humidity | DECIMAL(5,2) | 湿度（%） |
| light | INT | 光照强度（lux） |
| smoke | INT | 烟雾浓度 |
| collect_time | DATETIME | 采集时间 |

### 4.4 告警记录表 (alert_record)
| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT PK | 主键 |
| device_id | BIGINT | 关联设备ID |
| device_code | VARCHAR(50) | 设备编码 |
| alert_type | VARCHAR(50) | 告警类型 |
| alert_level | VARCHAR(20) | 告警级别（CRITICAL/HIGH/MEDIUM） |
| alert_content | VARCHAR(500) | 告警内容 |
| status | TINYINT | 处理状态（0未处理/1已处理） |
| create_time | DATETIME | 告警时间 |
| handle_time | DATETIME | 处理时间 |

## 五、AI Agent模块说明

### 5.1 数据模拟器 (DataSimulator)
- 定时为所有设备生成随机环境传感数据
- 可配置采集频率（默认10秒）
- 支持通过API或自然语言指令启动/停止

### 5.2 告警检测器 (AlertDetector)
- 自动检测异常数据并生成告警
- 温度>40°C → HIGH告警
- 温度<10°C → MEDIUM告警
- 湿度>80% 或 <20% → MEDIUM告警
- 烟雾>50 → CRITICAL告警

### 5.3 自然语言指令解析器 (NlpCommandParser)
- 支持指令："开始采集"、"停止采集"、"设置采集频率为N秒"、"系统巡检"
- 模糊匹配，容错解析

### 5.4 自动巡检 (AutoInspector)
- 检查设备在线状态
- 检查数据采集量
- 检查未处理告警
- 输出巡检报告与修复建议
